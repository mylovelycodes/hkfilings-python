# Code of Conduct

This project follows the [Contributor Covenant][cc] version 2.1.

In short: be respectful, assume good faith, focus on the work. Personal
attacks, harassment, and discriminatory language have no place here.

Maintainers are responsible for clarifying and enforcing this standard.
We may remove, edit, or reject comments, commits, code, issues, and
other contributions that are not aligned with the spirit of this code.

Report concerns to **conduct@hkfilings.app**. All complaints will be
reviewed and investigated promptly and fairly.

For the full text, see the Contributor Covenant page linked below.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
